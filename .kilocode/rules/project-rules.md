# กฎพิเศษสำหรับโปรเจ็คนี้

## 1. First Priority Rule
ทุกคำสั่งที่ผู้ใช้สั่งให้ทำ ต้องถูกพิจารณาเป็น first priority เสมอ โดยไม่ต้องสนใจว่าโค้ดเบสจะเป็นอย่างไร

## 2. Ask User Rule
ทุกข้อโต้แย้งและคำแนะนำให้ถามผู้ใช้เสมอ ก่อนลงมือทำ

## 3. Integration Rule
เมื่อติดตั้งสิ่งใดในโปรเจ็ค ให้ถือว่าต้องทำและปรับใช้ให้เข้ากับโปรเจ็คเสมอ

## 4. Mode Binding Rule
ตรวจสอบ custom code เพื่อเช็คว่าสิ่งที่ติดตั้งนั้นมีความเกี่ยวข้องกับ mode ใดหรือไม่ ถ้าเกี่ยวข้อง ให้ทำการผูกเข้ากับคอนแท็กเสมอ เพื่อให้เกิดการเรียกใช้งานหรืออยู่ในขอบเขตที่ mode นั้นๆ รับผิดชอบ

## 5. Workflow Rule
เมื่องานใดๆ ต้องกระทำซ้ำหรือซ้ำใน ให้สร้าง .kilocode/workflows/*.md เสมอ แล้วลงรายละเอียดสิ่งที่ต้องปฏิบัติตามลำดับ
