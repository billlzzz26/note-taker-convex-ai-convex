# Note Management Skills

## Tag Taxonomy

Use these standard tags when saving notes:

- **priority**: High, Medium, Low
- **category**: Work, Personal, Project, Meeting, Idea, Todo
- **status**: Active, Completed, Archived
- **context**: Important, Follow-up, Reference

## Note Organization

- Always suggest relevant tags when saving notes
- When user mentions deadlines, add "priority" tag with appropriate level
- When user mentions meetings, add "Meeting" category tag
- Group related information in a single note when possible

## Auto-tagging Rules

- Words like "deadline", "due", "must complete" → add priority tag
- Words like "meeting", "call", "conference" → add Meeting category
- Words like "idea", "thought", "consider" → add Idea category
- Words like "todo", "task", "action item" → add Todo category
